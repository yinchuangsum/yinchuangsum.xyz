# yinchuangsum-website

Personal CTO website — built with Astro, deployed on Cloudflare Pages.

## Language

**Now page**:
The homepage (`/`) showing current focus areas and what the owner is actively building.
_Avoid_: Home, landing page

**Blog post**:
A markdown content collection entry — hand-written technical deep dives on software architecture, engineering practices, and tools.
_Avoid_: Article, essay

**Project**:
A commercial product or open-source tool the owner has built. Displayed on the `/projects` page.

**Event**:
A speaking engagement or conference appearance. Displayed on the `/speaking` page.

**Training course**:
A live workshop or cohort-based program the owner delivers. Has a status (active/upcoming/past), cohort date, description, outline, and optional external link. Stored in the `education` content collection. Displayed on the `/education` page.

**Mentoring**:
One-on-one or small-group guidance offered by the owner. Described in a static profile section on the `/education` page. Not a content collection entry.

**Credibility marker**:
A past teaching, training, or mentoring affiliation used as social proof in the mentoring section. Displayed as a list item with institution name and short outcome. Not a content collection entry.
